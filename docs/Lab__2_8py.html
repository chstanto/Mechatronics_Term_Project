Detailed Description

This file defines two finite state machines to blink LEDs.

This file defines two finite state machines, the first of which blinks a virtual LED and the second of which modulates the brightness of LED A5 on the Nucleo microcontroller. The LED brightness modulation is in the form of a sawtooth wave.
