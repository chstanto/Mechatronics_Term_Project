Detailed Description

Sets up user interface task when called in spyder console.

This file is the main file on the front end of Lab 6. When this file is executed in the Spyder console window it will run the lab 6 user interface task. The task has been set up to run at an interval of 0.01 seconds. For more information please see Lab_6_UI.py .
