Detailed Description

This file runs TaskElevator for two asynchronous elevators.

This file defines motors, buttons, and limit switches required for two elevators to run simultaneously. Also defnined in this script are input parameters for the TaskElevator function along with the number of loops that should be run. Reference TaskElevator class for information on the elevator finite state machine.
