\doxysubsection{Detailed Description}
This file contains a class for obtaining angular data from the bno055 I\+MU module. 

This file acts as a driver for the bno055 inertial measurement unit. The driver uses I2C communication to write write settings to the I\+MU and read the appropriate registers. For this application we are setting the I\+MU to the N\+D\+OF fustion mode and retreiving the eularian pitch and roll values in radians.
