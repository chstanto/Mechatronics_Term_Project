\doxysubsection{Detailed Description}
This lab simulates the Vendotron machine. 

A finite state machine first requests that the user input change using number keys 0 -\/ 7, and at any point the user can request one of the four drinks by inputting the appropriate letter. The F\+SM then checks that the inputted funds are sufficient and either asks for more money or dispenses the requested drink. The machine then initializes again and prints the welcome message.
