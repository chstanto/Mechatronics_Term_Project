\doxysubsection{Detailed Description}
Sets up methods required to control a motor. 

The Motor Driver class allows the user to set up two motors using the {\bfseries{init}} method. This method assigns the appropriate pins, timers, and channels required by the motors. The class also includes the enable and disable methods which set the motors enable pin either to high or low. Finally the set duty method allows the user to input a desired motor duty cycle after it has been enabled.
