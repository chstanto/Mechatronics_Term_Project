\doxysubsection{Detailed Description}
This driver allows the nucleo to gather position data from the touch panel. 

This file defines the Touch\+\_\+\+Panel class which has five methods. Methods include init, Scan\+\_\+x, Scan\+\_\+y, Scan\+\_\+z, and get\+\_\+pos. By using the provided methods the user can determine if anything is in contact with the touch panel and the location of that contact.
